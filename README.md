# a_itogovi

## О проекте

**Энциклопедия космоса** — это интерактивный образовательный веб-проект, созданный для увлекательного изучения астрономии. Он предлагает пользователям погрузиться в мир планет, звезд, галактик и истории освоения космоса через статьи, интерактивные элементы и тесты.

## Особенности

*   **Интерактивная главная страница**: Динамические факты о космосе, обновляемые при каждом посещении.
*   **Обширная теоретическая база**: Подробные разделы, посвященные Солнечной системе, звездам, галактикам и истории космических исследований.
*   **Интерактивное исследование планет**: Изучайте каждую планету Солнечной системы с помощью детальных описаний и изображений.
*   **3D-модель Солнечной системы**: Визуализируйте движение планет в реальном времени с помощью интерактивной 3D-модели, реализованной на Three.js.
*   **Интерактивные созвездия**: Исследуйте известные созвездия, узнавая их названия и истории.
*   **Проверка знаний**: Закрепите полученные знания, пройдя тесты, реализованные через Google Forms.
*   **Адаптивный дизайн**: Удобный просмотр на любых устройствах благодаря адаптивной верстке.

## Технологии

Проект разработан с использованием следующих технологий:

*   **HTML5**: Структура веб-страниц.
*   **CSS3**: Стилизация и адаптивный дизайн.
*   **JavaScript**: Интерактивные элементы и динамический контент.
*   **Three.js**: Библиотека для создания 3D-графики в браузере (используется для 3D-модели Солнечной системы).

## Установка и запуск

Проект является статическим веб-сайтом и не требует специальной установки или серверной части. Для запуска:

1.  Клонируйте репозиторий:
    ```bash
    git clone https://github.com/Ваш_пользователь/a_itogovi.git
    ```
2.  Перейдите в директорию проекта:
    ```bash
    cd a_itogovi
    ```
3.  Откройте файл `index.html` в вашем веб-браузере.
